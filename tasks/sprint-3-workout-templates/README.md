# Sprint 3：训练模板（Workout Templates）

版本：v1.0  
状态：Ready

## Sprint 目标

交付完整的训练模板能力，让用户能够：

- 查看训练模板列表
- 创建训练模板
- 从动作库添加动作
- 配置目标组数、次数范围与休息时间
- 调整动作顺序
- 编辑已有模板
- 归档模板
- 在不影响历史训练的前提下长期维护模板

## 任务顺序

1. `S3-01-template-domain-and-contracts.md`
2. `S3-02-template-schema-and-migration.md`
3. `S3-03-template-repository.md`
4. `S3-04-template-list-screen.md`
5. `S3-05-create-template.md`
6. `S3-06-edit-template.md`
7. `S3-07-exercise-configuration-and-reordering.md`
8. `S3-08-template-archive.md`
9. `S3-09-sprint-exit-review.md`

每次只执行一个任务，完成 Review 并提交后再进入下一项。
