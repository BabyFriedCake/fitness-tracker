# Sprint 1 — Project Bootstrap

Version: v1.0  
Status: Ready

## Goal

Create a runnable Expo mobile workspace under `apps/mobile/` without implementing product features.

## Deliverables

1. pnpm workspace root
2. Expo + TypeScript mobile app
3. Expo Router placeholder navigation
4. SQLite bootstrap and migration runner
5. Lint, typecheck, and test commands
6. GitHub Actions CI

## Non-goals

- No Exercise Library implementation
- No Workout Template implementation
- No production database schema beyond the approved initial migration scaffold
- No final visual design
- No analytics, authentication, cloud sync, or subscriptions

## Execution order

1. `S1-01-workspace-and-expo.md`
2. `S1-02-code-quality.md`
3. `S1-03-router-shell.md`
4. `S1-04-sqlite-bootstrap.md`
5. `S1-05-ci.md`
6. `S1-06-bootstrap-review.md`

Each task should be completed and reviewed before moving to the next.
