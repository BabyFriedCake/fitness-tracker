# Prototype Specification Standard

Version: v1.0
Status: Approved

每个页面统一包含：Metadata、Screen Contract、Page Goal、Responsibilities、User Story、User Journey、Entry & Exit、Information Architecture、Wireframe、Component Inventory、Screen Data Dictionary、Visibility Rules、Interaction Matrix、State Machine、Data Dependency、Source of Truth、Navigation、Edge Cases、Analytics、Design Decisions、Acceptance Criteria、Future Extension、Out of Scope。

边界：
- 描述产品行为，不描述 React Native 实现。
- 引用 Domain Model，不展开 SQLite 字段。
- 一个页面只保留一份正式文件。
- 所有关键按钮、状态、异常必须可验证。
