# Prototype Review Checklist

- [ ] 页面目标、职责和用户故事明确
- [ ] Entry、Exit、Navigation 完整
- [ ] Wireframe 与信息层级完整
- [ ] 组件引用 Design System
- [ ] Data Dependency 与 Source of Truth 明确
- [ ] Loading、Empty、Normal、Error 等状态完整
- [ ] 重复点击、后台、恢复、数据异常已覆盖
- [ ] Analytics 与 Acceptance Criteria 可执行
- [ ] 与 PRD、Domain Model、Database、Design System 一致
