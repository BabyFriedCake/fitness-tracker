# Workout Companion Design System

## 产品定位

Mobile workout companion app。

目标体验：

像 Keep 一样的陪练体验。

用户主要关注：

-   当前动作
-   当前次数
-   当前状态
-   教练提示

## Layout

设备：

iPhone Mobile

设计原则：

-   大图片
-   少文字
-   高可读性
-   单手操作

## Typography

标题：

用于动作名称、训练阶段。

正文：

用于训练提示。

数字：

用于次数、倒计时。

## Components Style

Card:

-   圆角
-   清晰层级
-   简洁阴影

Button:

主要操作：

开始训练

继续训练

完成一组

Secondary:

暂停

跳过

## Interaction Style

状态驱动：

running

paused

resting

completed
