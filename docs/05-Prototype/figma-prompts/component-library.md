# Workout Companion Component Library

## Exercise Image Card

用途：

展示当前训练动作。

内容：

-   动作图片
-   动作名称
-   肌群

## Rep Counter

用途：

展示自动计数。

显示：

例如：

8 / 10

状态：

-   counting
-   completed

## Voice Coach Bubble

用途：

展示语音陪练状态。

示例：

🔊 第八次

## Rest Timer

用途：

组间休息。

显示：

30s

## Workout Controls

包含：

-   暂停
-   继续
-   下一动作

## Exercise List Item

显示：

-   图片
-   名称
-   组数
-   次数
