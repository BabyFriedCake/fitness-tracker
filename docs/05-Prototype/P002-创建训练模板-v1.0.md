# 1. 页面目标（Page Goal）

帮助用户创建一套可重复使用的训练模板。

模板描述的是**计划怎么练**，不是某一次真实训练。

---

# 2. 用户故事（User Story）

作为一名力量训练用户，

我希望提前配置好训练模板，

这样每次到健身房时，只需一键开始训练，而不用重新添加动作。

---

# 3. 用户入口（Entry）

进入方式：

- Today 首页点击「创建训练模板」
- 模板列表点击「新增模板」

退出方式：

- 保存模板返回模板列表

# 11. 边界情况（Edge Cases）

- 模板名称为空时禁止保存。
- 至少包含一个动作才能保存。
- 删除动作需确认。
- 返回时存在未保存修改，弹出确认框。

---

# 12. Design Decisions

- 用户只能从标准动作库选择动作，保证数据一致性。
- 默认采用统一组数和休息时间，用户可逐项编辑。
- 模板与真实训练完全分离，修改模板不会影响历史记录。

---

# 13. 验收标准（Acceptance Criteria）

- 能创建模板。
- 能添加多个动作。
- 能调整顺序。
- 能编辑组数、次数范围和休息时间。
- 能成功保存并用于开始训练。
