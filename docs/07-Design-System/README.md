# Fitness Tracker Design System

Version: v1.0  
Status: Approved  
Owner: Product Owner  
Last Updated: 2026-07-14

本目录定义 Fitness Tracker 的视觉、组件、交互和文案规范。

## 文件

- `design-system.md`：整体视觉与设计原则
- `components.md`：基础组件规范
- `workout-ui.md`：训练场景专用交互规范
- `content-style.md`：中文文案与数据展示规范
- `accessibility.md`：可访问性与易用性要求

## 设计方向

以专业训练工具风格为主，融合简洁的信息组织方式。

关键词：

- 专业
- 高效
- 清晰
- 数据优先
- 少动画
- 适合单手操作

## Future Experience

包含：

- Voice Coach 状态展示
- AI Coach Card
- Exercise Detail Layout
- History Analytics
